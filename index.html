SOBRE MIM

Olá, meu nome é Wendril Ferreira e estou cursando o curso técnico de Desenvolvimento de Sistemas no Senac Hub Academy. Tenho 23 anos e estou sempre aberto a aprender tudo relacionado à área de TI. Apesar de ter menos de um ano de curso, estou adorando aprender sobre Front-End e Back-End. Meu grande objetivo é entender como tudo funciona por trás de um site ou sistemas específicos, tanto na manipulação quanto na criação. Gosto muito dessa área e desejo muito crescer e trabalhar nela. Esta é minha primeira experiência com muitas tecnologias, e estou adorando o que estou aprendendo. Já adquiri conhecimentos em várias linguagens, como Python (minha primeira), HTML, CSS e JavaScript, além de Banco de Dados SQL e Análise de Requisitos..

EXPERIÊNCIAS

PYTHON

A linguagem Python foi a minha porta de entrada no mundo da informática. Com ela, aprendi o famoso “Hello World” e dei meus primeiros passos na programação. Aprendi sobre variáveis, comandos, ate mesmo criar uma calculadora e muitos outros exercícios que me ensinaram os fundamentos de um sistema.

HTML CSS

HTML e CSS me ensinaram como funciona e como criar toda a estrutura de um site. Nossa mentora nos mostrou como usar cores, modificar textos, formatar espaçamentos, criar figuras, adicionar imagens e até mesmo criar um simples botão. Aprendi tudo o que um site precisa para ter um ótimo design e infraestrutura, além de como interligar todos esses elementos.

JAVA SCRIPT

JavaScript é uma linguagem nova para mim, e ainda estou aprendendo sobre ela. No entanto, já percebi que ela é essencial para adicionar efeitos importantes e tornar o site totalmente funcional em diversos dispositivos.

SQL

SQL, ou Banco de Dados, foi uma experiência boa para mim, onde aprendi sobre diversos tipos de dados. Aprendi a criar um banco de dados, manipular dados, cadastrar clientes e produtos, e entender como o banco de dados funciona dentro de um sistema. Também aprendi a criar servidores, alterar dados e, principalmente, garantir a segurança do banco de dados. Foi uma experiência nova e reveladora, mostrando a importância dessa funcionalidade na gestão de dados
